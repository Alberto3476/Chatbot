# ctx

Este objeto se encuentra en `addAnswer` y `addAction`.

Este objeto puede contener datos como `from` (el número de teléfono por dónde se está comunicando el usuario) o si esperas una respuesta, también se puede usar la propiedad `body` para obtener la entrada del usuario.

```js
    const { addKeyword } = require('@bot-whatsapp/bot')

    const flowWelcome = addKeyword('hola').addAnswer('¿Como es tu mail?',null, async (ctx) => {
        console.log(ctx)
    })
```

---

### ¿Qué propiedades tiene el ctx?

Dentro del ctx puedes conseguir algunas de las siguientes propiedades

```js
    const { addKeyword } = require('@bot-whatsapp/bot')

    const flowWelcome = addKeyword('hola').addAnswer('¿Como es tu mail?',{ capture: true }, (ctx) => {
        console.log(ctx)

        const numeroDeWhatsapp = ctx.from 
        const mensajeRecibido = ctx.body 

    })
```

Te recomendamos revisar las propiedades que contiene el objeto `ctx` usando un `console.log`. Algunas de las propiedades también pueden pertenecer al `provider`.
